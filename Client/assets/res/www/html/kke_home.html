면접관2

개발하시면서 기술적으로 어려운 문제를 해결한 경험이 있으신가요? 맞닥뜨렸던 문제가 무엇이었는지, 해결을 위해 얼마나 깊이 있게 고민을 했는지 그 과정에 대해 이야기해 주세요.

운영 중이던 하이브리드 앱을 안드로이드 네이티브 앱으로 신규 개발하는 프로젝트를 맡고 있었는데 JSON 형태로 넘어오는 데이터값이 HTML 태그로 나열된 문자열이었어서 XML 데이터 바인딩에 오류가 계속 발생하였고 소스 코드로 HTML을 XML로 변환하는 방법을 찾으려 했지만 미흡한 실력으로 인해 이를 해결하지 못하고 결국 처리 방식을 변경하여 개발을 완료 했습니다.
변경한 개발 처리 방식은 XML에서 데이터 바인딩시에도 오류가 나지 않도록 데이터 값이 문자열로만 출력되는 인터페이스를 별도로 추가 생성하여 연결하는 것으로 해결하였습니다.
가장 최근에 진행했던 프로젝트라서 아직도 HTML을 XML로 변환하는 방법을 찾지 못한 상태이지만 추 후 방법을 모색하여 이를 꼭 해결할 계획입니다.
